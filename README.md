# TechnoMarket
Этот сайт представляет собой полноценный интернет-магазин электроники под названием «TechnoMarket». Вот его ключевые особенности и функциональность:

🧱 Общая структура

· Шапка – контакты, переключатель темы, кнопки входа/регистрации, логотип.
· Поиск – динамический поиск по товарам с автоподбором результатов.
· Навигация – главная, корзина, избранное, сравнение; выпадающие подменю для категорий.
· Карусель – автоматически переключающиеся баннеры с кнопками «вперед/назад» и точками.
· Категории и товары – сетка популярных категорий и карточек товаров (хиты продаж).
· Футер – контакты, ссылки, подписка на рассылку, иконки соцсетей, способы оплаты.

🛒 Корзина

· Добавление товара с количеством, изменение количества, удаление.
· Подсчёт итоговой суммы, бесплатная доставка, поле для промокода (TECHNO10).
· Возможность продолжить покупки или оформить заказ (имитация оформления с модальным окном).

❤️ Избранное (Wishlist)

· Добавление/удаление товаров, сохранение в localStorage.
· Отдельная страница со списком избранного и кнопкой «В корзину».

🔁 Сравнение товаров

· Добавление до 4 товаров, отображение характеристик в таблице.
· Удаление отдельных товаров или очистка всего списка.

🌙 Тёмная тема

· Переключатель в шапке, сохраняющий выбор пользователя.

🎨 Дизайн и интерактивность

· Адаптивная сетка (CSS Grid и Flexbox).
· Анимации при наведении (увеличение карточек, показ кнопки быстрого просмотра).
· Всплывающие уведомления (toast) при добавлении в корзину/избранное.
· Модальное окно с деталями товара (характеристики, цена, кнопки действий).
· Кнопка «Наверх» при скролле.

📦 Хранение данных

· Все данные (корзина, избранное, сравнение) сохраняются в localStorage.
· Товары и категории жёстко заданы в JavaScript (имитация базы данных).

🔧 Дополнительный функционал

· Параллакс-эффект на фоне героя.
· Валидация email в форме подписки.
· Имитация входа/регистрации через модальные окна.
· Загрузка контента страниц (корзина, избранное, сравнение) динамически.

💡 Технологии

· HTML5 / CSS3 – кастомные стили, переменные, ключевые кадры.
· JavaScript (ES6) – вся логика на чистом JS без внешних фреймворков.
· Font Awesome 6 – иконки.
· Animate.css – для части анимаций.

⚠️ Особенности реализации

· Полностью фронтенд-прототип – все действия (покупка, промокод, регистрация) имитируются и не отправляют реальные запросы на сервер.
· Картинки товаров используются из Unsplash (примеры).
· Код самодостаточен – достаточно открыть index.html в браузере, чтобы всё заработало.
